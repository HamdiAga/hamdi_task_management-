اكسل للمبرمجين والتقنيين: دوال لا تعرفها ستجعلك تدهش مديرك

10 يونيو 2026 حمدي فواز أبو حاتم Office Tools

معظم المبرمجين ينظرون إلى Excel كأداة "للمحاسبين"، لكنهم لا يدركون أن Excel هو في جوهره محرك قواعد بيانات مع واجهة رسومية. عندما تتعامل مع Excel كالمبرمج لا كالمستخدم العادي، تكتشف إمكانيات مذهلة. سأشاركك أقوى الدوال والتقنيات التي ستجعلك تنجز في دقائق ما يأخذ زملاءك ساعات.

1. لماذا Excel صديق المبرمج الخفي؟

  • يحتوي على لغة برمجة كاملة (VBA) ومحرك صيغ قوي (DAX في Power Pivot).
  • يدعم Power Query للتعامل مع البيانات الضخمة ومصادر متعددة.
  • يمكن ربطه مباشرة بقواعد البيانات (SQL Server, MySQL, PostgreSQL).
  • يدعم JSON و XML.
  • مع Excel Labs (إضافة مايكروسوفت)، يمكنك استخدام Python داخل Excel!

2. دوال متقدمة ستغير تعاملك مع البيانات

الدالةماذا تفعلمثال عملي
XLOOKUPبحث فائق، بديل VLOOKUP المحدودابحث عن سعر منتج من جدول آخر
FILTERتصفية بيانات بناءً على شرطاعرض كل الطلبات المعلقة فقط
SORT / SORTBYترتيب ديناميكيرتّب المنتجات حسب المبيعات تنازلياً
UNIQUEاستخراج القيم الفريدة فقطما المدن التي وصلت إليها الشحنات؟
TEXTJOINدمج نصوص مع فاصلاجمع أسماء المندوبين في خلية واحدة
LAMBDAتعريف دوال مخصصة بدون VBAأنشئ دالة تحسب الضريبة 15%

3. LAMBDA – اكتب دوالك الخاصة كالمبرمج

هذه أقوى ميزة للمبرمجين. تستطيع تعريف دالة مخصصة قابلة لإعادة الاستخدام، تماماً مثل function في الكود:

=LAMBDA(price, price * 1.15)

لحفظها كدالة دائمة: Name Manager > New > Name: "AddVAT" > Refers to: =LAMBDA(price, price * 1.15). الآن تستخدمها في أي خلية: =AddVAT(A1). هذا يشبه تماماً كتابة دالة في JavaScript.

4. Power Query – ETL داخل Excel

Power Query هي أداة ETL (Extract, Transform, Load) مدمجة في Excel. تستطيع:

  • دمج ملفات CSV متعددة: إذا كان لديك تقارير يومية منفصلة، فادمجها بنقرة واحدة.
  • استيراد JSON مباشرة: إذا كان لديك API يرجع JSON، فاستورده مباشرة إلى Excel.
  • تنظيف البيانات: إزالة الفراغات، تغيير الأنواع، دمج الأعمدة.
  • الاتصال بقاعدة بيانات: SQL Server، MySQL، وحتى MongoDB.

لماذا يهمك كمبرمج: Power Query يشبه كتابة LINQ في C# أو Pipeline في Python. الخطوات مسجلة وقابلة لإعادة التشغيل، مما يعني أنك تبني "برنامجاً" داخل Excel.

5. ربط Excel بقاعدة بيانات MySQL مباشرة

  1. Data > Get Data > From Database > From MySQL Database.
  2. أدخل Server (localhost أو IP)، Database name، Username، Password.
  3. اختر الجدول أو اكتب استعلام SQL مخصص.
  4. Excel سيحمل البيانات مباشرة، مع إمكانية تحديثها بنقرة واحدة.

هذا يعني أن مديرك يفتح Excel، يضغط Refresh، ويرى أرقام المبيعات الحية من قاعدة بيانات متجرك!

6. أتمتة Excel باستخدام Python (عبر Excel Labs)

أضف إضافة Excel Labs من متجر الإضافات، وستحصل على محرر Python داخل Excel:

import pandas as pd

# قراءة بيانات من Excel
df = xl("A1:D100", headers=True)

# تجميع البيانات
summary = df.groupby('category')['sales'].sum().reset_index()

# كتابة النتيجة في Excel
xl("F1", summary)

النتيجة: قوة Python الكاملة داخل Excel، بدون تصدير أو استيراد.

من تجربتي: في أحد المشاريع، كان المدير يقضي 3 ساعات كل اثنين في دمج 7 ملفات مبيعات يدوياً. بنيت له Power Query يجمعها تلقائياً، مع Pivot Table محدثة بنقرة Refresh. العملية صارت تأخذ 30 ثانية. هذا النوع من الأتمتة هو ما يجعل المبرمج "ساحراً" في نظر الإدارة.

خلاصة

Excel ليس مجرد جداول. هو محرك قواعد بيانات، لغة برمجة، وأداة ETL في آن واحد. تعلم XLOOKUP، LAMBDA، Power Query، وربطه بقاعدة البيانات. هذه المهارات تجعلك تحل مشاكل الأعمال في دقائق، وتدهش مديرك.
