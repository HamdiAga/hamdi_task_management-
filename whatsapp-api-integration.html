واتساب هو التطبيق الأول في الخليج. عملاؤك يستخدمونه يومياً، ويتوقعون أن يجدوا متجرك هناك. دمج WhatsApp Business API بموقعك ليس رفاهية، بل ضرورة تنافسية. سأشرح لك خطوات الدمج بالتفصيل، من التسجيل إلى إرسال أول رسالة تلقائية، مع أمثلة عملية من متاجر حقيقية.

2. البدء مع WhatsApp Cloud API

أسهل طريقة للدمج هي Cloud API المقدمة من Meta مباشرة. خطوات التسجيل:

  1. اذهب إلى developers.facebook.com وأنشئ تطبيقاً من نوع Business.
  2. أضف منتج WhatsApp، واختر Create App.
  3. ستحصل على معرف تطبيق (App ID) ورقم هاتف اختباري، و Access Token دائم.
  4. للإنتاج، ستحتاج لتوثيق حساب Business Manager.

3. إرسال رسالة من الـ Backend (Node.js)

const axios = require('axios');

const WHATSAPP_TOKEN = 'your-access-token';
const PHONE_NUMBER_ID = 'your-phone-number-id';

async function sendWhatsAppMessage(to, templateName, parameters) {
    try {
        const response = await axios.post(
            `https://graph.facebook.com/v21.0/${PHONE_NUMBER_ID}/messages`,
            {
                messaging_product: 'whatsapp',
                to: to, // 966512345678
                type: 'template',
                template: {
                    name: templateName,
                    language: { code: 'ar' },
                    components: [{
                        type: 'body',
                        parameters: parameters // [{ type: 'text', text: 'حمدي' }, ...]
                    }]
                }
            },
            {
                headers: {
                    'Authorization': `Bearer ${WHATSAPP_TOKEN}`,
                    'Content-Type': 'application/json'
                }
            }
        );
        console.log('تم الإرسال:', response.data);
        return response.data;
    } catch (error) {
        console.error('خطأ في الإرسال:', error.response?.data || error.message);
    }
}

// مثال: إرسال تأكيد طلب
await sendWhatsAppMessage('966512345678', 'order_confirmation', [
    { type: 'text', text: 'حمدي' },
    { type: 'text', text: '12345' },
    { type: 'text', text: '299 ريال' }
]);

4. سيناريوهات عملية للتكامل

  • تأكيد الطلب: عند إنشاء طلب جديد، أرسل رسالة فورية: "مرحباً حمدي، تم استلام طلبك رقم 12345 بقيمة 299 ر.س. سنخبرك عند الشحن."
  • تحديث حالة الشحن: عند تغيير حالة الطلب إلى "تم الشحن"، أرسل رسالة: "تم شحن طلبك رقم 12345. شركة الشحن: أرامكس. رقم التتبع: 67890."
  • تذكير السلة المهملة: إذا أضاف العميل منتجات للسلة ولم يكمل الشراء، أرسل تذكيراً بعد ساعتين: "منتجاتك في السلة بانتظارك! أكمل الشراء الآن واحصل على خصم 5%."
  • طلب التقييم: بعد 3 أيام من تسليم الطلب، أرسل: "كيف كانت تجربتك مع منتج X؟ قيّمنا من 1 إلى 5."

5. استخدام Webhook لاستقبال ردود العملاء

لجعل المحادثة تفاعلية، تحتاج لاستقبال رسائل العملاء عبر Webhook:

// Laravel: routes/api.php
Route::post('/webhooks/whatsapp', function(Request $request) {
    $entry = $request->input('entry');
    
    foreach ($entry as $e) {
        foreach ($e['changes'] as $change) {
            if ($change['field'] === 'messages') {
                foreach ($change['value']['messages'] ?? [] as $message) {
                    $from = $message['from']; // رقم المرسل
                    $text = $message['text']['body']; // نص الرسالة
                    
                    // خزّن في قاعدة البيانات
                    WhatsAppMessage::create([
                        'from' => $from,
                        'message' => $text,
                        'received_at' => now()
                    ]);
                    
                    // رد تلقائي بسيط
                    // (يحتاج لموافقة القالب من Meta)
                }
            }
        }
    }
    
    return response()->json(['success' => true]);
});

6. تكاليف WhatsApp Cloud API

أول 1000 محادثة شهرياً مجانية. بعد ذلك، التكلفة حوالي 0.05-0.10 ريال لكل محادثة (حسب البلد). المحادثة هي نافذة 24 ساعة بين العميل والنشاط التجاري. هذا يعني أن 1000 طلب شهرياً قد يكلفك 50-100 ريال فقط.

من تجربتي: في أحد متاجر العطور، أضفنا إشعارات واتساب للطلبات والشحن. انخفضت استفسارات "أين طلبي؟" بنسبة 70%، وارتفع تقييم المتجر في غوغل بسبب سرعة التواصل. التكلفة كانت أقل من 80 ريالاً شهرياً.

خلاصة

دمج واتساب API هو أحد أفضل الاستثمارات التقنية لمتجرك. ابدأ بـ Cloud API المجانية، وابنِ إشعارات الطلبات والشحن أولاً. كلما وفرت على العميل عناء السؤال، كلما زادت ثقته وولاؤه لمتجرك.
